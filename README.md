# AutoBump

Данный скрипт умеет загружать капчу от бота **[Server Monitoring](https://discord-server.com)** на сайт **[RuCaptcha](https://rucaptcha.com)**.

  - Загрузка изображения
  - Получение ответа на пример
  - Отправка ответа боту
  - Магия
